# gotemplate

Template golang application to make project setup ready and steady!

In this gotemplate project structure, tests , ci/cd pipelines, Dockerfiles etc. everything is pretermined to make it easy to use.

## Getting Started

Run dependencies
```
make up
```

Run the code
```
make run
```

Run unit tests
```
make test-unit
```

Run tests
```
make test
```

Run linter
```
make lint
```

Generate mocks
```
make mockgen
```
